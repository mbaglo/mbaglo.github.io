# mbaglo.github.io

This repository contains the source for the academic website of **Chinyere Gloria Mbaka**.

## Pages

- `index.html` — home page with profile summary, research areas, and links.
- `about.html` — biography and background.
- `research.html` — research focus and current projects.
- `blog.html` — blog landing page.

## Purpose

The site presents academic background, research interests, and professional information in a simple static format suitable for GitHub Pages.

## Deployment

This site is intended to be published with GitHub Pages from this repository.
